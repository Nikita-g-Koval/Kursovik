3. Исходные данные для создания программной системы:

3.1. Перечень решаемых задач:}

\renewcommand\labelenumi{\theenumi)}

\begin{itemize}
\item проанализировать систему для оценки и контроля знаний;
\item разработать концептуальную модель системы для оценки и контроля знаний;
\item спроектировать программно-информационную систему для оценки и контроля знаний;
\item реализовать программно-информационную систему тестирования средствами графических библиотек;
\item протестировать программно-информационную систему для оценки и контроля знаний.
\end{itemize}

{\parindent0pt
  3.2. Входные данные и требуемые результаты для программы:}

\begin{enumerate}
\item Входными данными для программно-информационной системы являются: данные имеющихся тестов, данные о зарегистрированных пользователях, ответы на вопросы теста, данные администратора.
\item Выходными данными для программно-информационной системы являются: графическое отображение вопросов тестов; данные создании/изменении тестов; данные о новых зарегистрированных/добавленных администратором пользователях; данные о изменении паролей пользователей; данные результатов пройденных тестов.
\end{enumerate}
